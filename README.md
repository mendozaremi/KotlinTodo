# KotlinTodo
Kotlin Todo app using RecyclerView

Project 2 Google Android black+ development program

- Add Todo's on RecyclerView List
- Delete Todo's that are checked

## Technology used
Kotlin, RecyclerView, CheckedBox, EditText, Buttons
